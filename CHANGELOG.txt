OverBuddy has been updated and now supports Steam!
- Improved stability by only allowing one instance of the app to run.
- Fixed an issue preventing some users from enabling Steam integration.

Other changes in this update:
- Added support for Steam installations of Overwatch.
- Added an option in Settings to enable the Overwatch debug console.
- Fixed various issues and made improvements.

Thank you for using OverBuddy! This major update brings important changes for a more reliable and complete experience.
